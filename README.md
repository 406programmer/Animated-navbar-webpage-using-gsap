Animated Navbar Webpage using GSAP

A simple animated navigation menu built with HTML, CSS, and GSAP (GreenSock Animation Platform). When the user clicks the menu icon, an animated sidebar slides in. Clicking the close icon hides it again with smooth animation.

🔹 Demo

No live demo provided yet — you can open the project locally to see the animation.

✨ Features

Smooth navbar animation using GSAP.

Sidebar menu slides in from the right.

Staggered animated title and close icon.

Lightweight & easy to customize.
